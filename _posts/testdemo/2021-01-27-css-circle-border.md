## 문제

the avatar's border is rounded so that it appears as a circle  

## 첫 번째 풀이

### 정답코드  

```css
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Avatar</title>
    <style>
    .avatar {
        border:2px solid gray;
        height:150px;
        border-radius: 50%;
        width:150px;
    }
    </style>
  </head>
  <body>
    <img class="avatar" src="https://goo.gl/khGCrk" alt="avatar" />
  </body>
</html>
```

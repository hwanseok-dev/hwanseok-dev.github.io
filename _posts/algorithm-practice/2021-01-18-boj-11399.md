## 첫 번째 풀이 : 그리디

### 알고리즘

SJF를 사용하면 됩니다. 가장 짧은 값을 먼저 처리하면 됩니다.  

### 정답코드  

```cpp
#include <bits/stdc++.h>
using namespace std;

using ll = long long;
using pii = pair<int, int>;

const int MAX = 100'001;

int arr[MAX];

int dx[4] = { 0,0,1,-1 };
int dy[4] = { 1,-1,0,0 };

int n, m, k;

int main()
{
	//freopen("input.txt", "r", stdin);
	ios_base::sync_with_stdio(0); cin.tie(0);
	cin >> n;
	int ans = INT_MAX;
	for (int i = 0; i < n; i++) {
		cin >> arr[i];
		ans = min(ans, arr[i]);
	}
	cout << ans * n;
	return 0;
}

```

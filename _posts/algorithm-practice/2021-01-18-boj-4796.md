## 첫 번째 풀이 : 그리디

### 알고리즘

V일 중에 P일이 몇 번 반복할 수 있는지, 그리고 며칠이 남는지 생각해봅니다.  
남은 일 수와 L의 최소값을 더해야하는 것에 주의합니다.  

### 정답코드  

```cpp
#include <bits/stdc++.h>
using namespace std;

using ll = long long;
using pii = pair<int, int>;

const int MAX = 101;

int n, m, k;

int main()
{
	//freopen("input.txt", "r", stdin);
	//ios_base::sync_with_stdio(0); cin.tie(0);
	int i = 1;
	while (true) {
		scanf("%d %d %d", &n, &m, &k);
		if (!(n|m|k)) {
			break;
		}
		ll ans = n * ll(k / m);
		ans += min(k % m, n);
		printf("Case %d: %lld\n", i, ans);
		i++;
	}
	return 0;
}
```

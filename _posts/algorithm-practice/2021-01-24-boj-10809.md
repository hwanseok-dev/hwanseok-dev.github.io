## 첫 번째 풀이

### 정답코드  

출력시 join을 써도 되고 \*를 써도 됩니다.  

```python
import sys

# sys.stdin = open("input.txt", "r")

ans = ['-1'] * 26
s = input()
for i, c in enumerate(s):
    idx = ord(c) - ord('a')
    if ans[idx] == '-1':
        ans[idx] = str(i)

print(' '.join(ans))
# print(*ans)

```
